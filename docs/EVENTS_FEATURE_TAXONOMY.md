# Master Feature Taxonomy (~200+ features in 12 categories)

## A. Event Creation & Management

1. Quick event creation (< 1 minute for simple events)
2. Detailed event builder (multi-step wizard)
3. Event types: In-person, Virtual, Hybrid
4. 11+ categories: Music, Sports, Business, Food, Tech, Community, Religious, Cultural, Education, Entertainment, Nightlife
5. Event templates (birthday, wedding, conference, meetup, concert, workshop)
6. Themed event pages (visual themes/animations like Partiful)
7. Custom branding (banner, logo, colors)
8. Multi-day events with separate sessions/tracks
9. Recurring events (daily, weekly, monthly, custom)
10. Event cloning/duplication
11. Draft/publish workflow with preview
12. Co-hosting with multiple organizers
13. Event series / programs
14. Custom URL slugs
15. Multi-language descriptions (English + Swahili)
16. Location picker with map
17. Virtual event link (Zoom, Google Meet, or built-in)
18. Agenda/schedule builder (multi-session events)
19. Speaker/performer management
20. Sponsor listing and placement

## B. Ticketing & Registration

1. Free RSVP events
2. Paid ticket events
3. Multiple ticket tiers (General, VIP, VVIP, Early Bird)
4. Promo/discount codes
5. Group discounts
6. Time-based ticket availability (on-sale / off-sale dates)
7. Quantity limits per order (min/max)
8. Waitlist when sold out
9. Invite-only / access code tickets
10. Approval-based registration (organizer screens attendees)
11. Custom registration forms
12. Add-ons (parking, food, merchandise)
13. QR code mobile tickets (rotating/live QR for anti-fraud)
14. SMS ticket delivery (for feature phones)
15. Ticket transfer to another person
16. Ticket gifting
17. Refund policies (configurable: full, partial, none)
18. Dynamic/demand-based pricing
19. Reserved seating with interactive seat map
20. Donation / pay-what-you-want option
21. Anti-scalping (non-transferable, rotating QR like DICE)
22. Ticket bundles/packages
23. Season pass / multi-event pass
24. Family/group registration
25. "Interested" lightweight RSVP (no commitment, like Facebook)

## C. Payment & Monetization

1. M-Pesa integration (STK push) — critical for TZ/KE
2. Tigo Pesa / Airtel Money integration
3. Credit/debit card processing
4. Multiple currency (TZS, KES, UGX, USD)
5. Fee absorption option (organizer vs buyer pays)
6. Organizer payouts (mobile money or bank)
7. Real-time sales tracking
8. Revenue reports and analytics
9. Tax/receipt generation
10. Refund processing
11. Split payments (group events)
12. Payment plans / installments
13. Platform commission structure
14. Tip/donate to organizer or performer
15. Sponsor payment management
16. Apple Pay / Google Pay

## D. Discovery & Search

1. Location-based "near me" discovery
2. Map view of events
3. Category/genre browsing
4. Date filters (Today, Tomorrow, This Weekend, This Week, This Month)
5. Price filters (Free, Paid, Price range)
6. Keyword search with autocomplete
7. Personalized recommendations (interests, history, social graph)
8. "Friends going" social proof
9. Trending/popular events
10. "Just Announced" new events
11. Followed organizers' events feed
12. Events from user's groups/communities
13. "Events like this" suggestions
14. Curated collections/editorial picks
15. Artist/performer alerts
16. Calendar view of saved/RSVPed events
17. Nearby venue browsing
18. "Happening Now" live events

11. Organizer profiles with event history

16. Community/group events integration
17. Potluck/signup lists (who's bringing what)
18. Playlist collaboration (Spotify integration)

## F. Organizer Tools

1. Dashboard with real-time analytics
2. Attendee management (list, export, filter)
3. QR code scanner app for check-in
4. Team/staff roles (admin, door staff, volunteer, co-host)

12. Tracking pixels (Facebook, Google Analytics)

15. Event budget tracking

## G. Communication

1. Confirmation notifications (push + email + SMS)
2. Reminders (1 day, 1 hour, 15 min before)

4. Event change notifications (time, venue, cancellation)

6. Direct messaging between attendees

8. Post-event thank you and survey
9. Push notifications for followed organizers' new events
10. WhatsApp integration for reminders

## H. Media & Content

1. Event cover images and galleries
2. Event videos / trailers
3. Live streaming within event
4. Event stories (short-form content)
5. Post-event photo gallery (crowdsourced)
6. Post-event video highlights
7. Speaker/performer profiles with media
8. Venue photos and 360-degree view
9. Event flyer/poster designer (in-app)
10. AR photo frames/filters for events

## I. Calendar & Time

1. Add to Google/Apple/Outlook Calendar
2. Calendar view of all events (RSVPed, interested, saved)
3. Time zone display and conversion
4. Recurring event patterns
5. Conflict detection (overlapping events)
6. Agenda view for multi-session events
7. Event countdown timer

## J. Maps & Location

1. Venue location on map
2. Directions/navigation
3. Uber/Bolt ride integration
4. Parking information
5. Nearby food/accommodation suggestions
6. Indoor venue maps for large events

## K. Virtual/Hybrid Event Features (from Hopin/Luma)

1. Built-in video conferencing
2. Breakout sessions / parallel tracks
3. Speed networking (random 1-on-1 video matching)
4. Virtual expo/sponsor booths
5. Live polls and Q&A
6. Event-wide + session chat
7. Recording and replay
8. Simulcast to YouTube/social media
9. Virtual backgrounds
10. Digital business card exchange

## L. East Africa-Specific Features

1. M-Pesa / mobile money as PRIMARY payment
2. Swahili-first UI with English toggle
3. SMS-based interactions (feature phone users)
4. Low-bandwidth optimization (compressed images, lazy loading)
5. Offline event details caching
6. WhatsApp as primary sharing channel
7. Local categories (Bongo Flava, Gospel, Ngoma, Sherehe)
8. Cultural event types (Harusi/wedding, Msiba/funeral, Harambee/fundraiser, church events)
9. Agent/kiosk ticket sales (physical distribution)
10. USSD fallback for ticket purchase
11. Multi-currency (TZS default, KES, UGG)
12. Local venue database
13. Community contribution model (Harambee)

## Top 10 Critical Differentiators for TAJIRI

| Priority | Feature | Why |
|----------|---------|-----|
| 1 | M-Pesa/mobile money native | Primary payment method in East Africa |
| 2 | WhatsApp-first sharing | Primary social channel in the region |
| 3 | "Friends going" social proof | Leverage TAJIRI's existing social graph |
| 4 | Cultural event types | Harusi, Harambee, Sherehe, Msiba |
| 5 | SMS ticket delivery | Feature phone support |
| 6 | Swahili-first UI | Already the pattern in TAJIRI |
| 7 | Low-bandwidth / offline | Data is expensive |
| 8 | Community/group events | Tie into TAJIRI's existing groups |
| 9 | Creator monetization | Organizer tips, subscriptions |
| 10 | Event wall/social feed | Make events social (Partiful-style) |

---

The current TAJIRI events module has ~15 of these ~200 features. The biggest gaps are: no auth on endpoints, no mobile money, no social features (friends going, comments, sharing), no discovery algorithm, no virtual events, no organizer tools, and no media/content support.
